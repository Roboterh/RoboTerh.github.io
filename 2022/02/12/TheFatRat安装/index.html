TheFatRat安装

2022-02-12
环境安装-kali

下载地址

点击这里跳转github项目

  1. 下载命令 git clone https://github.com/Screetsec/TheFatRat.git
  2. 可以使用proxychains挂代理提升速度

安装步骤

  1. fatrat
  2. setup.sh
  3. powerfull.sh

三个文件需要可执行权限

  • sudo ./setup.sh

安装中：

选择 2

是否创建快捷方式，选择是

  • sudo ./fatrat

成功安装，可以执行操作了
